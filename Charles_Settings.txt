Настройка Charles 

1. Подключаем телефон к компьютеру.

2. Начальная настройка:
Заходим во вкладку Help - SSL Proxying - Install Charles Root Certificate on a Mobile Device Proxying.
В появившемся окне указан IP и порт для телефона.
Идем в настройки девайса - wifi.
Выбираем нашу Wifi сеть - настройки - редактировать - расширенные настройки
Прокси сервер - Вручную
и ставим ip и порт, save.
Как только пойдет трафик (с телефона зайдем в интернет), 
Charles оповестит о том, что с устройстрайства начался прогон трафика.
В появившемся окне чарлика жмем Allow

3. Чтобы снифить защищенный траффик, установим SLL сертификат:
На телефоне заходим на сайт 
https://chls.pro/ssl
Download
Profiles device manager
выбираем скачанный сертификат - install
Теперь нужно довериться сертификату
trust settings

Идем в чарлике на вкладку proxy - ssl poxy settings
Галочку Enable SSL Proxy
И ввести 
*.*
*
[]
